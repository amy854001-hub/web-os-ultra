# 🚀 WebOS Ultra

An ultra-lightweight, high-performance web-based operating system simulator built entirely in a single, optimized file. WebOS Ultra brings a smooth, minimalist desktop environment straight to your web browser.

## ✨ Features

* **Desktop Environment:** Full desktop interface featuring custom SVG system icons, desktop shortcuts, and a clean taskbar.
* **Window Management:** Draggable, resizable, and minimizable application windows with smooth window mechanics.
* **Core Simulator Apps:**
  * 🖥️ **Terminal / Command Prompt:** Run system simulation commands.
  * 📝 **Text Editor:** Create and edit local text files.
  * 🎨 **SVG Viewer / Canvas:** Render custom vector graphics directly inside the simulator.
  * ⚙️ **Settings Panel:** Customize theme colors, background wallpapers, and view system stats.
* **Lightweight Architecture:** Optimized code footprint for fast loading times and zero lag, designed to run smoothly anywhere.

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Vanilla JavaScript
* **Assets:** Custom inline SVG graphic assets

## 🚀 Quick Start

Since the entire operating system simulator is bundled inside `webOS.html`, running it is incredibly simple:

1. Clone or download this repository.
2. Open `webOS.html` in any modern web browser (Chrome, Edge, Firefox, or Safari).
3. That's it! No installation, web servers, or complex setup required.
